jQuery Playground

Trying using the various elements on this page as a playground to do DOM manipulations with.

Here are some ideas of things you should try:

  • Changing the content of an element
  • Adding or removing elements
  • Changing the background of an element
  • Adding or removing classes to an element
  • Changing the size of an element
  • Adding event listeners on objects, and having them react appropriately
  • Playing with forms, and their behaviours. (Try manipulating form contents with code!)
